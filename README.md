This application is a social network based on Twitter where the user can be followed by other users, tweet, retweet, like tweets etc.

Functionalities

* Create user profile
* Edit user profile
* Remove user profile
* Folow users
* Feed
* Create tweet
* Delete tweet
* Like tweets
* Update tweets
* Search tweets ou users
* trending topics list
* Retweet

Stack
- Environment
	* Docker
	* Docker Compose
- API
	* Ruby on Rails
	* Search
		-- Elasticsearch
	* Background Jobs
		-- Redis
		-- Sidekiq
- Client
	* React
		-- Axios
		-- Redux
