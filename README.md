Social network based on Twitter where the user can be followed by other users, tweet and retweet, like tweets etc.

functionalities

-Create user profile
-Edit user profile
-Remove user profile
-Folow users
-Feed
-Create tweet
-Delete tweet
-Like tweets
-Update tweets
-Search tweets ou users
-trending topics list
-Retweet
